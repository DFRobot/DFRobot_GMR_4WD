定制四轮控制机器人 开放API说明

使用xbee作为无线通信设备，波特率为9600。

协议说明：
-----------------------------------------------------------------------------------------	
|    字头	设备	桢长度	命令字	内容(这里是3字节)	校验和	结束字		|
-----------------------------------------------------------------------------------------
|  0x55 0xAA	0x10	0x03	0x03	0x11	0x80	0x80	sum	0x0D	0x0A 	|
-----------------------------------------------------------------------------------------

支持的命令字：
0x03		控制小车左右电机
0x55		请求行程数据(返回命令字为0x57数据串)		
0x56		请求行程数据(直接返回行程字符串)	

0x57		命令0x55返回的数据


校验和计算方法：
从字头开始的所有命令做加法，取最低位的一个字节作为校验和
参考代码如下：

//
uint8_t calcChecksum (uint8_t *theBuf) {
	uint8_t leng = theBuf[3] + 5;
	uint8_t sum = 0;
	for (int i=0; i<leng; i++) {
		sum += theBuf[i];
	}
	return sum;
}


电机速度绝对值控制指令(共11字节)：
---------------------------------------------------------------------------------------------------------	
|	字头		设备地址	桢长度	命令字	方向	速度1	速度2	校验和	结束字		|
---------------------------------------------------------------------------------------------------------
|	0x55 0xAA	0x10		0x03	0x03	DIR	0x80	0x80	sum	0x0D	0x0A 	|
---------------------------------------------------------------------------------------------------------
速度的单位是cm/s，最大值是255cm/s


DIR为电机旋转方向，0表示正转，1表示反转，其真值表如下所示：
---------------------------------
|		正转	反转	|
---------------------------------
|	电机1	0x00	0x01	|
---------------------------------
|	电机2	0x00	0x10	|
---------------------------------


请求行程命令字0x55格式(共11字节)：
---------------------------------------------------------------------------------------------------------	
|	字头		设备地址	桢长度	命令字				校验和	结束字		|
---------------------------------------------------------------------------------------------------------
|	0x55 0xAA	0x10		0x03	0x55	0x00	0x00	0x00	sum	0x0D	0x0A 	|
---------------------------------------------------------------------------------------------------------


命令字0x55返回的数据的格式(命令字0x57, 共16字节)：
---------------------------------------------------------------------------------------------------------------------------------	
|   字头   	设备	桢长度	命令字	左轮行程(4 byte)		右轮行程(4 byte)		校验和	结束字		|
---------------------------------------------------------------------------------------------------------------------------------	
|  0x55 0xAA	0x10	0x08	0x57	0x00	0x00	0x00	0x00	0x00	0x00	0x00	0x00	sum	0x0D	0x0A 	|
---------------------------------------------------------------------------------------------------------------------------------	
左右轮行程为四字节有符号整型数，单位是m(米)
